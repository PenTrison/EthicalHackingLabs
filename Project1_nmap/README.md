# Project 1: Network Scanning Lab

## Objective
To scan a vulnerable virtual machine and identify open ports and services.

## Tools
- Kali Linux
- Nmap
- VMware
- Metasploitable

## Methodology
1. Set up Kali and Metasploitable in VMware.
2. Used Nmap to discover open ports.
3. Analyzed running services.

## Results
- Found multiple open ports including FTP, SSH, Telnet, HTTP, SMB.

## Security Recommendations
- Disable unused services.
- Use firewall rules.
- Apply updates and patches.
