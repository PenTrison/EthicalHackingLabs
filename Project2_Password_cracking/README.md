# Project 2: Password Cracking Lab

## Objective
To demonstrate how weak passwords can be cracked using dictionary attacks.

## Tools
- Kali Linux
- John the Ripper
- RockYou wordlist

## Methodology
1. Created test password hashes.
2. Used John the Ripper with RockYou wordlist.
3. Observed cracking results.

## Results
- Weak passwords cracked easily.
- Strong passwords resisted cracking.

## Security Recommendations
- Use long complex passwords.
- Enable multi-factor authentication.
