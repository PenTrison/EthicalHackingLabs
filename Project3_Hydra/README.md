# Project 3: Hydra FTP Brute Force Lab

## Objective
To simulate a brute force login attack on a vulnerable FTP service.

## Tools
- Kali Linux
- Hydra
- Metasploitable VM

## Methodology
1. Created a password list.
2. Used Hydra to brute force FTP login.
3. Identified valid credentials.

## Results
- Successfully discovered FTP login credentials.

## Security Recommendations
- Use strong passwords.
- Limit login attempts.
- Enable account lockout policies.
